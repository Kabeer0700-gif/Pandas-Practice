# 🐼 Pandas Practice

A hands-on repository for practicing data analysis and manipulation using the **Pandas** library in Python. Covers real-world datasets with practical notebook-based exercises in data wrangling, exploration, and feature extraction.

---

## 📁 Repository Structure

```
Pandas-Practice/
│
├── coutries.ipynb            # Data analysis on world countries dataset
├── Countries.csv             # Countries dataset
│
├── feature_Extraction.ipynb  # Feature extraction techniques with Pandas
├── anime.csv                 # Anime dataset used for feature extraction
│
└── README.md
```

---

## 📓 Projects

### 1. 🌍 Countries Data Analysis — `coutries.ipynb`

Exploratory data analysis on a world countries dataset. Explores global statistics and demographic/economic features across different countries.

**Dataset:** `Countries.csv`

**Key Steps:**
- Loading and inspecting country-level data
- Handling missing values and data types
- Filtering, sorting, and grouping with Pandas
- Descriptive statistics and trend analysis
- Visualizing country-wise comparisons

---

### 2. 🎌 Feature Extraction — `feature_Extraction.ipynb`

Demonstrates feature extraction and data manipulation techniques using the Pandas library, applied to an anime dataset.

**Dataset:** `anime.csv`

**Key Steps:**
- Data loading and initial exploration
- Extracting and transforming features from raw data
- String and categorical data manipulation
- Creating derived features
- Data filtering and aggregation

## 🚀 Getting Started

### Prerequisites

Install the required libraries:

```bash
pip install pandas numpy matplotlib seaborn jupyter
```

### Run the Notebooks

```bash
# Clone the repository
git clone https://github.com/Kabeer0700-gif/Pandas-Practice.git

# Navigate to the project directory
cd Pandas-Practice

# Launch Jupyter Notebook
jupyter notebook
```
